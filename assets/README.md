# Custom coin art (optional)

The game ships with original drawn mascots. To use a real logo for any coin,
drop an image here named after the character id:

  bonk.png   wif.png   pop.png   chad.png   pnut.png   fart.png

Square PNGs (e.g. 256×256, transparent background) look best. If a file is
present it's used everywhere (menu picker + in-game); if not, the built-in
mascot shows. No code changes needed — just add the file and redeploy.

Only add art you have the rights to use.
